Dom Diffrent Width

1-innerWidth: The innerWidth property only works for window its always give the width of the window , this property don't work for any html block or elemnt, it give undefine if we find the innerWidth of any html/block
2-offsetWidth: The offsetWidth property returns the total amount of space of an element horizontally, including the width of the content, scrollbars (if any), padding, and border. but don't calculate the margin.
3-clientWidth:The clientWidth returns how much an element takes up horizontally, including padding but not including the border, margins, or scrollbars.
4-scrollWidth:The scrollWidth returns the actual size of the content, regardless of how much of it is currently visible.
It returns inner div width + inner div padding + outer div padding.
5-getBoundingClientRect:getBoundingClientRect gives an oject which is having the information about the location and dimension of the element.
