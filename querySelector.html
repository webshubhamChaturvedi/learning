IMPORTANT: when using methods like getElementsByClassName, getElementsByTagName, and getElementsByName, the result is a live collection. If the DOM changes , the collection will automatically update to reflect those changes. querySelector and querySelectorAll return static collections, and they won't automatically update if the DOM changes after the selection.
